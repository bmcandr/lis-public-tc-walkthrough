=== Download Input Files

The input files are available on the {url-lis-testcases}[LIS Testcase website]. While within your `$WORKING_DIR`, download and unpack the compressed tar file containing the inputs for Step 5:

[#download-step1]
[source,bash,subs="attributes"]
----
% curl -O {url-lis-testcase-files}{step5-filename}
% tar -xzf {step5-filename} -C .
----
